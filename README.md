## Overview
This module handles user authentication for the RevPlay application.

## Features
- User Registration
- User Login
- Change Password
- Forgot Password (Security Question based)

## Tech Stack
- Java 21
- Oracle 23c Free (PDB)
- JDBC
- IntelliJ IDEA

## How to Run
1. Start Oracle Database (FREEPDB1)
2. Create required tables (USERS, LOGIN_HISTORY)
3. Add `ojdbc11.jar` to classpath
4. Run `MainApp`
